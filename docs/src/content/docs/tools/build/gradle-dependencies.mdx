## Overview

Display the dependency tree for a module, helping you understand what libraries your project uses and identify version conflicts.

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `projectDir` | `string` | Yes | — | Absolute path to the Android project root |
| `module` | `string` | No | `:app` | Module to show dependencies for |
| `configuration` | `string` | No | — | Configuration to show (e.g., `debugCompileClasspath`) |

## Examples

> Show the dependency tree for my app

> Show debugCompileClasspath dependencies for the :app module

> Show releaseRuntimeClasspath dependencies

<Aside type="tip">
  Use a specific configuration like `debugCompileClasspath` to narrow down the output — the full tree can be very long.
</Aside>
