## Overview

Extract and list every permission declared in an APK's manifest. Useful for auditing before release.

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `apkPath` | `string` | Yes | — | Absolute path to the APK file |

## Examples

> List permissions in my release APK

> Check what permissions my debug APK requests

<Aside type="caution">
  Third-party libraries can add unexpected permissions. Always review before publishing.
</Aside>
