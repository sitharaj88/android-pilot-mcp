## Overview

Retrieve comprehensive device details including model, manufacturer, Android version, API level, screen density, resolution, and available storage.

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `deviceId` | `string` | No | — | Target device serial |

## Examples

> Get info about my connected device

> Show device model, OS version, and screen density
