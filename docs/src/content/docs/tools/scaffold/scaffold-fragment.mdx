## Overview

Generate a Fragment class with boilerplate code and an optional XML layout file.

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `projectDir` | `string` | Yes | — | Absolute path to the Android project root |
| `fragmentName` | `string` | Yes | — | Fragment class name (e.g., `ProfileFragment`) |
| `packageName` | `string` | Yes | — | Package name |
| `module` | `string` | No | `app` | Module name |
| `layout` | `boolean` | No | `true` | Generate a corresponding XML layout file |
