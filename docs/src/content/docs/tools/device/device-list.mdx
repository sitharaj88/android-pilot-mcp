## Overview

Show all connected physical devices and running emulators with their serial numbers and connection status.

## Parameters

No parameters required.

## Examples

> List my connected Android devices

> Show all connected devices and emulators

## Related Tools

- [Device Info](/android-pilot-mcp/tools/debug/device-info/) — Get detailed info about a specific device
- [Start Emulator](/android-pilot-mcp/tools/device/emulator-start/) — Start an emulator if none are running
