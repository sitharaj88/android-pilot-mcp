---
title: Stop App
description: Force stop an application on a connected Android device.
---

## Overview

Force stop a running application. Equivalent to swiping it away from recents.

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `packageName` | `string` | Yes | — | Application package name to stop |
| `deviceId` | `string` | No | — | Target device serial |

## Examples

> Force stop com.example.myapp

> Stop my running app
