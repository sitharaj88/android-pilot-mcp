## Overview

Gracefully stop a running emulator by its device ID.

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `deviceId` | `string` | Yes | — | Device serial (e.g., `emulator-5554`). Use `device_list` to find it. |

## Examples

> Stop the running emulator

> Stop emulator-5554
