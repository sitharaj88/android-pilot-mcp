title: List App Permissions
description: List all permissions for an installed app, showing granted and denied status.
---

## Overview

View every permission declared by an app and whether it's currently granted or denied.

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `packageName` | `string` | Yes | — | Application package name |
| `deviceId` | `string` | No | — | Target device serial |

## Examples

> List all permissions for com.example.myapp

> Show which permissions are granted for my app
