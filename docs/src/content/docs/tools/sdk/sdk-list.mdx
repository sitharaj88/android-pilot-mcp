## Overview

Show installed SDK packages or browse all available packages including platforms, system images, build tools, and more.

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `installed` | `boolean` | No | `true` | If true, show only installed packages. If false, show all available. |

## Examples

> List my installed SDK packages

> Show all available SDK packages
