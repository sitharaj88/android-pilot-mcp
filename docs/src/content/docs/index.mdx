## What Can It Do?

<CardGrid stagger>
  <Card title="Build & Lint" icon="setting">
    Gradle builds, task runner, dependency trees, project clean, and Android Lint
    analysis — all from your editor.
  </Card>
  <Card title="Device Management" icon="laptop">
    List, create, start, and stop emulators. Install APKs, launch apps, manage
    permissions, WiFi ADB, and file transfers.
  </Card>
  <Card title="Debugging" icon="magnifier">
    Logcat filtering, screenshots, screen recording, UI hierarchy dumps, device
    info, and shell commands.
  </Card>
  <Card title="Scaffolding" icon="document">
    Create full Android projects with Compose or XML. Scaffold activities,
    fragments, and Compose screens from templates.
  </Card>
  <Card title="APK Analysis" icon="information">
    Inspect APK size, versions, SDK targets, DEX references, file listings,
    manifests, and permissions.
  </Card>
  <Card title="Intents & Deep Links" icon="rocket">
    Send intents, broadcast events, and test deep link URIs on connected
    devices.
  </Card>
  <Card title="SDK Management" icon="list-format">
    List installed and available SDK packages. Install platforms, system images,
    and build tools.
  </Card>
</CardGrid>

## Quick Start

```bash
npx android-pilot-mcp
```

Add to your Claude Code config (`.mcp.json`):

```json
{
  "mcpServers": {
    "android-pilot": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "android-pilot-mcp"]
    }
  }
}
```
